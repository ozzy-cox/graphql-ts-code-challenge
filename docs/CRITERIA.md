# Evaluation Criteria

The jury will evaluate the criteria below:
 
### GraphQL Schema Quality
- Reflecting the domain
- Advanced usage
- Documentation

### Architecture
- Reflecting the domain
- Abstraction
- Separation of concern

### Code Quality
- Linter
- Formatter
- Strict type safety
- Stick to a convention
- Being easy-to-read
- Sustainability

### Testing
- Unit testing
- E2E testing
